## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Como rodar o projeto

### Usando Docker (recomendado)

1. Instale o [Docker](https://www.docker.com/) e o [Docker Compose](https://docs.docker.com/compose/).
2. Clone o repositório e acesse a pasta `backend`.
3. Copie o arquivo `.env.example` para `.env` (ajuste se necessário).
4. Execute o comando:

```bash
docker-compose up --build
```

O backend NestJS e o banco PostgreSQL serão inicializados automaticamente.
A API estará disponível em `http://localhost:3000`.
O Swagger estará em `http://localhost:3000/api`.

### Rodando manualmente (sem Docker)

1. Instale o Node.js (versão 20+) e o PostgreSQL localmente.
2. Configure o arquivo `.env` conforme o exemplo em `.env.example`.
3. Instale as dependências:

```bash
npm install
```

4. Execute as migrações do banco:

```bash
npx prisma migrate dev
```

(Caso de erro de SSh CERTIFICATE ao rodar o comando "npx prisma migrate dev"  exculte no terminal que está: $env:NODE_TLS_REJECT_UNAUTHORIZED=0)

5. Inicie o backend:

```bash
npm run start:dev
```

O backend estará disponível em `http://localhost:3000`.
